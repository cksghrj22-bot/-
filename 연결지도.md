## 연결이 완성되면 도는 루프

```
조회수(유튜브 API) ─┐
씨앗(음성 한마디) ──┼→ 조합엔진(prompts/01) → 대본/카피 → 밤 공장(본진) → 9/16/23시 발행
지식(노션+옵시디언) ┘                                      ↑
                    반응 데이터가 다시 조합엔진으로 ────────┘ (자가발전)
```
