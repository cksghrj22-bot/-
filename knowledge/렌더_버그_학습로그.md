# 🐞 렌더 버그 학습로그 — 같은 원인 두 번 안 겪는다 (2026-07-16)

> 이찬호 지시: "원인을 자꾸 학습해야 될 거 아니야." 여기 적힌 건 **코드로 이미 고쳐졌고**, 증상→원인→수정을 남겨 다음 세션이 재발 안 시킨다.

## 1. BGM이 조용히 사라짐 (가장 오래 헤맴)
- **증상**: `--bgm`을 줬는데 영상에 음악이 없음. 에러도 없음(조용한 무음).
- **진짜 원인**: `shorts/proof.py`가 `probe_duration`을 **import 안 함** → `_valid_bgm()` 안에서 `probe_duration(path)` 호출이 **NameError** → `except`로 잡혀 **무조건 False** → `bgm_pool`이 유효 BGM을 버림 → 렌더는 음악 없이 진행.
- **수정**: `from .render import render, probe_duration` (import 추가). + `probe_duration`을 ffprobe 실패 시 ffmpeg 파싱 폴백(간헐 실패 방어). + `render_batch`에 "BGM 못 읽으면 경고" 추가(조용한 무음 방지).
- **교훈**: `except Exception` 광범위 캐치가 NameError까지 삼켜 버그를 숨겼다. 조용한 실패는 반드시 경고를 남긴다.

## 2. BGM 파일 손상 (curl 다운로드)
- **증상**: bgm mp3가 `0 channels / Duration N/A`로 디코드 불가.
- **원인**: 공개공유 안 된 드라이브 파일을 curl로 받으면 **HTML/에러 페이지**가 받힘(로그인 벽). 그걸 mp3로 착각.
- **수정**: `get_file_permissions`로 `anyone: reader` 확인된 파일만 curl. 유효 파일 = `bgm_piano.mp3`(478KB, 29.9s, CreatorOS 피아노). 받은 뒤 `ffmpeg -i`로 채널/길이 검증 필수.

## 3. 교보손글씨가 대체폰트로 바뀜
- **증상**: 자막·제목 글자체가 매번 달라짐(나눔/노토로).
- **원인**: config 폰트 `Kyobo Handwriting 2019`가 **클라우드에 미설치** → libass가 대체. 새 세션마다 폰트 유실(커밋 안 돼 있어서).
- **수정**: `assets/fonts/KyoboHandwriting2019.ttf` 커밋 + `install_fonts.sh`(세션시작 설치) + 프롬프트북 절차 추가. `fc-match "Kyobo Handwriting 2019"` → KyoboHandwriting2019 매칭 확인.
- **교훈**: 세션은 기억 0. 폰트·B롤·결정은 **채팅이 아니라 리포지토리에** 남겨야 다음 세션이 안 까먹는다.

## 4. 아웃트로가 잘려 안 보임
- **증상**: 마지막 "SNS에 일기…" 아웃트로가 화면에 안 뜸.
- **원인**: 나레이션+BGM에 `-shortest` → 영상이 나레이션 길이로 잘리는데, 아웃트로는 배경(나레이션+tail) 길이에 찍혀 **잘림**.
- **수정**: `render.py` `-shortest` 제거 + `-t <배경길이>`로 고정 + 나레이션 `apad`로 tail까지 오디오 유지.

## 5. 교보손글씨가 또 나눔으로 뒤바뀜 + 검사기가 틀린 폰트를 통과시킴 (2026-07-16 재발)
- **증상**: config `style_preset_mind`의 자막·제목 폰트가 `Kyobo Handwriting 2019`가 아니라 `NanumSquareRound`로 바뀌어 있었고, 그 상태로 렌더된 4편이 나눔으로 구워짐. 그런데 검사기는 "✅ PASS"를 냄.
- **진짜 원인 (3중)**:
  - (a) config가 어느 시점에 Kyobo→Nanum으로 조용히 바뀜(근거 없는 재유추).
  - (b) 검사기가 **`NanumSquareRound`가 있어야 통과**하도록 짜여 있어 틀린 폰트를 정답으로 알고 통과시킴.
  - (c) **가장 은밀한 진짜 원인**: config를 Kyobo로 되돌려도 폰트명을 **`Kyobo Handwriting 2019`(공백 포함)**으로 쓰면 libass가 이 패밀리를 못 찾아 **DejaVuSans→한글은 중국어 WenQuanYiZenHei로 폴백**한다. 폰트 파일의 내부 패밀리명은 **`KyoboHandwriting2019`(공백 없음)**이라, ASS/config도 공백 없는 이 이름을 그대로 써야 실제로 매칭된다. `fc-match`는 퍼지매칭이라 공백판도 찾아주지만 **libass fontselect는 못 찾는다** — fc-match 통과 ≠ 실렌더 통과.
- **검증 실증** (ffmpeg `-vf ass` fontselect 로그): `(Kyobo Handwriting 2019, 700, 0) -> DejaVuSans-Bold` ❌ vs `(KyoboHandwriting2019, 400/700, 0) -> /root/.fonts/KyoboHandwriting2019.ttf` ✅.
- **수정**: (1) config 폰트 = **`KyoboHandwriting2019`**(공백 없음) 고정. (2) 검사기 4번을 둘로: 4a ASS 텍스트에 KyoboHandwriting2019, **4b ASS를 재렌더해 libass가 실제로 DejaVu/WenQuanYi/Noto로 폴백하는지** 로그로 검사(폴백이면 FAIL). (3) 4편 전부 재렌더. (4) `_폰트` 주석에 공백 금지 명시.
- **교훈**: 폰트는 **ASS 텍스트가 아니라 실제 구워진 픽셀**로 검증해야 한다. `fc-match`나 config 문자열이 맞아도 libass가 조용히 폴백하면 딴 글씨가 나온다. 그리고 이찬호 확정값(교보손글씨)은 config·검사기·규격문서 3곳이 항상 같아야 한다.

## 6. BGM이 아웃트로에서 꺼짐 — 피아노 페이드아웃이 영상 꼬리에 걸림 (2026-07-16)
- **증상**: 음성 speed 1.1→1.0으로 내리자 영상이 길어지며(27.4s→30.5s) 02·04가 검사기 "BGM tail RMS -89dB"로 FAIL. (01·03은 길이가 달라 우연히 통과 — 정렬 의존이라 불안정.)
- **원인**: bgm_piano.mp3(29.9s)가 **자체 앞 2초 페이드인·뒤 3초 페이드아웃**. `-stream_loop`으로 반복해도 그 조용한 head/tail이 주기적으로 재발 → 영상 꼬리(아웃트로 2.6초)가 그 조용한 구간에 걸리면 무음처럼 됨.
- **수정**: 피아노 **본체 2~27초(연속 음악)만 잘라 `bgm_piano_loop.mp3`(25s)** 로 루프 → head/tail 항상 음악(-35~-43dB). 4편 재렌더 전부 PASS.
- **교훈**: 루프 BGM은 **페이드 없는 본체만** 써야 길이가 바뀌어도 안전하다. 새 CreatorOS 곡도 앞뒤 페이드 있으면 본체만 트림해서 loop본으로 만든다.

## 근본 재발방지 (이번에 넣음)

## 7. 카드뉴스 글자 사이 뻥뻥 벌어짐 (2026-07-20)
- **증상**: 캐러셀 한글 텍스트의 공백이 비정상적으로 넓게 벌어짐("둥근    얼굴"). 글 다 깨져 보임.
- **원인**: CSS `font-family:"Pretendard","Noto Color Emoji",sans-serif` — Pretendard @font-face가 없어 로드 실패 → 다음 폰트 Noto Color Emoji가 **공백(U+0020)을 이모지 너비로 렌더**. 이모지 폰트를 텍스트 폰트 바로 뒤에 두면 발생.
- **수정**: 텍스트 폰트를 **설치된 한글폰트 우선**으로, 이모지 폰트는 **맨 뒤**로. 예: `font-family:"NanumSquareRound","Pretendard","Apple SD Gothic Neo",sans-serif,"Noto Color Emoji"`. 이 환경 설치 한글폰트: NanumSquareRound·BMJUA·PoorStory·KyoboHandwriting.
- **교훈**: 이모지 폰트는 **항상 font-family 맨 마지막**. 그 앞에 실제 로드되는 텍스트 폰트(로컬 설치본)를 둬야 공백·한글이 정상. @font-face(원격 woff2)는 렌더 시 로드 실패 가능 → 로컬 설치 폰트 폴백 필수.
- 재사용 템플릿: `shorts/cards/card_carousel_template.js` (Chano's Formula 캐러셀 — 사진배경+다크오버레이+👉리스트+✓팁).
