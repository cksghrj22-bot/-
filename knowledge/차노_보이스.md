# 차노 보이스 프로파일 (차노화의 정의)

노션 정본(블로그 2편·쇼츠 채널·브리핑 기록)에서 추출한 문체·사고 패턴. 모든 생성 콘텐츠는 이 프로파일을 통과해야 "차노화"된 것이다.

## 핵심 사고 패턴 (콘텐츠의 뼈대)

1. **재정의**: 통념의 단어를 뒤집어 다시 정의한다.
   - 숱치기 = 양 덜기 (X) → 공간 만들기 (O)
   - 볼륨 = 숱 (X) → 모근의 각도 (O)
   - 색 = 정하는 것 (X) → 스며드는 것 (O)
   - 레이어드 = 층 (X) → 움직임 (O)
2. **작은 차이의 확대**: "같은 가위질이라도 무슨 생각으로 대느냐" — 비행기 방향 비유처럼, 미세한 관점 차이가 결과 전체를 가른다.
3. **판정형 질문으로 마무리**: "당신은 기술을 팝니까, 서사를 팝니까?" / "당신의 디자이너는, 무드를 읽나요?" — 시청자가 자기 답을 내리게 한다. (2026-07-20 원복: 되묻기 마무리는 유지. 앞서 '어색' 지적은 텍스트가 아니라 일레븐랩스 발음('읽나요' 오독) 문제였음 → 06 규약 발음사전으로 해결.)
4. **업의 철학화**: 미용 기술 이야기가 반드시 삶/일의 철학으로 확장된다 (도제식 비판, 파이프라인 마인드, 가치 vs 가격).
5. **이질 결합**: 미용 밖의 것(카멜커피, 비행기, 옷장)을 끌어와 미용을 설명한다.

## 문체 규칙

- 존댓말 기본, 짧은 단문. "~예요/~죠/~합니다" 혼용.
- 훅은 공감 질문 또는 단정적 반전: "숱 쳤는데 더 지저분해진 적 있죠?" / "손님은 커트를 사러 오지 않습니다"
- 고객에게 실행 가능한 한 줄을 준다: "'숱 쳐주세요' 말고 '가볍게 움직이게 해주세요'"
- 브랜드 선언문으로 닫을 수 있다: "앳나운은 숱을 치지 않습니다. 공간을 디자인합니다."
- 금지: 과장 수식어 남발, 전문용어 자랑, 할인/영업 멘트.

## 세계관 상수

- 미션: **한 문장이 한 권의 책이 된다** — 한마디 → 씨앗 → 제안카드 → 원자 콘텐츠 → 지식 라이브러리 → 책.
- 운영 3단계: ①연결(한마디×자산×이질 개념) ②변환(1지식→다중 포맷) ③발신(국내→글로벌).
- 포맷 역할: 쇼츠·카드뉴스 = 원자·훅(입구) / 블로그 = 요리된 깊이(풀코스) / 책 = 총서.
