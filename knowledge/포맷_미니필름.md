# 🎞️ 포맷: 미니필름 (멀티씬 스토리텔링) — 2026-07-15 이찬호 지시로 신설

> 메시지 하나를 장면 3~4개가 ¼씩 나눠 짊어진다. 장면 하나는 근거, 장면들의 연결이 이야기.
> 쇼츠보다 길어도 되고(45~90초), 쇼츠 규격으로 압축해도 된다.

## 구조 공식 (¼ 법칙)

| 씬 | 역할 | 메시지 지분 |
|---|---|---|
| 씬1 | 설정 — 일상의 평범한 순간 (누구나 아는 장면) | ¼ 공감 |
| 씬2 | 반복 — 같은 행동의 디테일 클로즈업 (이 사람은 이걸 매일 한다) | ¼ 신뢰 |
| 씬3 | 반전 — 예상 밖 장면/단어 (재정의가 터지는 지점) | ¼ 각성 |
| 씬4 | 귀결 — 처음 장면으로 돌아오되 의미가 바뀐 컷 | ¼ 여운 |

나레이션은 한호흡(단일 합성), 각 씬 전환은 문장 전환과 일치시킨다. BGM은 씬3 직전에 반 박자 쉼.

## 이찬호 촬영 수칙 (폰 하나로 충분)

1. 한 장면 = **10~15초 고정샷** (팬/줌 금지, 폰 벽·컵에 기대서라도 고정)
2. 같은 장면을 **넓게 1번 + 가까이 1번** (조립 때 선택지가 두 배)
3. 밝은 쪽을 등지지 말 것 (창문은 얼굴 앞에)
4. 소리는 안 써도 됨 (나레이션·BGM이 덮음) — 찍을 때 말해도 OK
5. 찍은 날 드라이브 「앳나운_영상/미니필름_원본」에 던져놓기 — 파일명 그대로 OK, 분류는 Claude가

## 시리즈 제안 (이찬호의 요즘에서 나온 것)

### A. 「나의 요즘」 — 쉬는 날 편 (첫 편 추천)
- 씬1 알람 끄고 커튼 여는 손 / 씬2 아침 운동 한 동작(이미 자산 있음: 하이록스) /
  씬3 수첩에 일기 쓰는 손 클로즈업 (**반전: 미용사가 아니라 기록하는 사람**) / 씬4 살롱 문 여는 뒷모습
- 메시지: "돈보다 먼저 하는 투자" 실사판 — 말이 아니라 하루로 증명
- 이미 있는 자산: 운동(VID_20260626 시리즈) → **새로 찍을 건 씬1·3·4 세 컷뿐**

### B. 「손의 하루」
- 씬1 아침 손 씻기 / 씬2 빗 잡는 손 / 씬3 가위 대신 **수첩 잡은 손** / 씬4 손님 머리가 흔들리는 완성 컷
- 메시지: 매일 쌓은 시간이 손끝에 (기존 검증 문장의 영상화)

### C. 「한 단어 하루」 — 사전 실사판 (사전 시리즈와 연동)
- 예: 「공간」 — 씬1 아침 창문 열기 / 씬2 지하철 틈 사이 / 씬3 숱치기 가위질 / 씬4 흔들리는 머리
- 메시지: 살롱의 단어가 하루 전체에서 보인다 — 단어당 1편, 사전 쇼츠와 짝으로 발행

### D. 「먹는 날」 (쉬는 날 먹방이 아니라)
- 씬1 혼자 먹는 백반 / 씬2 젓가락질 클로즈업 / 씬3 수첩 메모 "맛있다 말고 왜 맛있는지" / 씬4 살롱에서 같은 눈으로 머리를 보는 컷
- 메시지: 관찰이 직업이 되는 사람 — "감각 있다"는 말의 정체 (키워드 C라인 연결)

## 파이프라인

이찬호 촬영 → 드라이브 업로드 → Claude가 씬 선별·조립안(어느 컷을 몇 초, 어떤 문장과) 제안 →
이찬호 글 확정 → 한호흡 나레이션 + 씬 전환 동기화 렌더 → 시안 → 발행 결정.
렌더러 멀티씬 지원(장면 이어붙이기)은 첫 촬영분 도착에 맞춰 확장한다 (proof.py — 씬 리스트 입력).

## 임베디드 문법 (매미킴TV 참조 — 2026-07-15 추가)

**복잡성 상한 (이찬호 경고 반영)**:

- 롱폼 임베디드(2~5분): **최소 4컷 이상** (이찬호 규칙 2026-07-15). 자유 편집 — 라이브러리에서 스토리를 조립.
