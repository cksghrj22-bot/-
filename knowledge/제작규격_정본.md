## 실행 명령 (정본)

```
python3 -m shorts.proof <대본폴더> --out <출력폴더> --preset style_preset_mind \
  --only NN --broll <원본영상> [--broll-start 초] --workdir <캐시폴더>
```

## 원칙 재확인

- 쇼츠 1편 = 고유 원본 1개 (배경 재사용 금지)
- 시안은 항상 "발행본 아님" 명시, 고객 얼굴 노출 시 발행 전 확인
- 산출물 보고 = 저장 위치 + 링크 필수
- 연결 장벽 발견 즉시, 우회와 동시에 "이찬호 1분 절차"를 같은 턴에 제시

이찬호가 글(스크립트)을 쓰면, Claude는 최고의 프로듀서·촬영감독·편집자로서 아래를 **계산해서 제안**한다.
스크립트만 있는 제안은 미완성 산출물로 취급한다.

1. **배경 영상 지정**: 드라이브 실자산 중 1개(1편=원본 1개) + 사용 구간(초). 후보 2개까지, 각각 이유.
2. **단어↔장면 조화**: 대본의 핵심 단어(재정의)가 화면의 어떤 순간과 만나는지 — 훅(0~3초)에 무슨 장면,
   반전("아니에요")에 무슨 장면, 결론에 무슨 장면. 말과 그림이 같은 것을 가리켜야 한다.
3. **3부류 시청자 계산**:
   - 까다로운 시청자: 0~3초 훅의 시각적 이유 (스크롤을 멈출 프레임)
   - 합리적인 시청자: 중반의 근거 장면 (주장을 눈으로 확인시키는 컷)
   - 시간 아끼는 시청자: 총 길이·자막 속도·결론 선행 여부 (30~40초, 말 끝나면 자막 즉시 전환)
4. **제목·썸네일 프레임**: 몇 초 지점을 썸네일로 쓸지 + 제목 문구.
