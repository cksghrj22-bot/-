# 🎬 쇼츠 제작 규격 정본 (2026-07-14 확정 — 까먹기 금지, 개정은 이찬호만)

모든 코드방 세션은 렌더 전에 이 문서를 따른다. 소스코드 반영 위치: `shorts_config.json`의 `style_preset_mind`.

## 화면 규격 (채널 정본 — 레퍼런스: 「디자이너의 색」 「결국 마인드 차이예요」)

- 캔버스: **1080×1920 (9:16) 고정**
- 영상 배치: 원본을 **4:5(1080×1350)로 확대 크롭** 후 중앙 배치 — 가로·세로 원본 모두 동일 (폭맞춤 레터박스는 화면이 작아져서 기각됨, 2026-07-14)
- 위아래는 **순수 검정 밴드** (각 285px). 검증법: 밴드 휘도 YAVG=16
- 화면 전체 느낌: 영상 위에 **반투명 블랙 45%** (dim) — 글씨가 주인공

## 글자 (2026-07-14 이찬호 확정)

- **제목**: 푸어스토리(Poor Story), 105px, 노랑(&H0000D7FF), 검정 외곽선, **상단 검정 밴드 안에 배치**
- **자막**: 감자꽃(Gamja Flower), 88px, 흰 글씨 + **검은 박스(불투명 85%)**, 영상 중앙~하단부
- **대사 규칙: 화면에 최대 2줄.** 대본 단계에서 한 세그먼트 ≤26자로 끊는다
- 줄바꿈이 어색하면 대본의 쉼표 위치를 조정 (박스 폭은 글자에 따라 자동)

## 보이스 (일레븐랩스 — 차노 프로페셔널 클론, 31분 원본)

- 설정 B 확정: stability 0.42 / similarity 0.85 / style 0.15 / speaker_boost ON / 1.1배속
- 문맥 스티칭(previous_text/next_text) 필수 — 줄별 합성이어도 끊기지 않음
- 띄어쓰기·쉼표를 대본에 정확히 — TTS가 쉼표 위치에서 쉰다
- TTS 캐시: 같은 문장+설정은 재합성 안 함 (workdir 고정 시)

## 실행 명령 (정본)

```
python3 -m shorts.proof <대본폴더> --out <출력폴더> --preset style_preset_mind \
  --only NN --broll <원본영상> [--broll-start 초] --workdir <캐시폴더>
```

## 원칙 재확인

- 쇼츠 1편 = 고유 원본 1개 (배경 재사용 금지)
- 시안은 항상 "발행본 아님" 명시, 고객 얼굴 노출 시 발행 전 확인
- 산출물 보고 = 저장 위치 + 링크 필수
- 연결 장벽 발견 즉시, 우회와 동시에 "이찬호 1분 절차"를 같은 턴에 제시
