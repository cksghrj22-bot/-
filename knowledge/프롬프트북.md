## 2. 명령어 시트 (Claude가 실행하는 정본 명령)

```bash
# 연결 진단 (세션 시작마다)
python3 -m pipeline check

# 시안 렌더 (마인드 스타일 정본)
python3 -m shorts.proof content/shorts/<날짜> --out content/shorts/<날짜>/보이스_시안 \
  --preset style_preset_mind --only NN --broll <원본영상경로> [--broll-start 초] \
  --workdir <스크래치>/proof_work    # ← TTS 캐시 재사용 (재렌더 몇십 초)

# 드라이브 업로드 (영상 → _최신_바로보기)
python3 -m shorts.gdrive upload <영상.mp4> --folder-id 1m3yfOIZknA-YZu5WjUtUUA9W-QJQLxsj

# 드라이브 인증이 처음이면 (링크+코드 → 폰에서 허용 한 번)
python3 -m shorts.gdrive auth

# 유튜브 일부공개 업로드 (파이썬 3줄)
python3 -c "from shorts import upload_youtube as yt; c=yt.load_credentials('secrets/youtube.json'); \
print(yt.upload('<영상.mp4>', yt.build_metadata('<제목>', description='<설명>', privacy='unlisted'), c))"

# 테스트 (커밋 전)
python3 -m unittest discover -s tests
```

## 4. 오늘(7/14) 삽질에서 나온 규칙 — 반복 금지

1. **환경 선택이 전부다**: 대본공장 환경 아니면 일레븐랩스 403. 세션 열 때 확인.
2. **키는 값 복사 대신 JSON 파일 첨부**로 받기 (폰 복사는 실패율 높음).
3. **기기 인증엔 「TV 및 제한된 입력 장치」 유형 클라이언트** — 데스크톱 유형은 invalid_client.
4. API 403 = 십중팔구 **콘솔에서 해당 API 미활성화** → activationUrl 눌러 「사용」.
5. 연결 장벽 발견 즉시, 우회와 **동시에** 이찬호 1분 절차 제시 (왕복 줄이기).
6. 대사 최대 2줄 (세그먼트 ≤26자) — 대본 단계에서 지킨다.
7. 산출물 = 위치+링크 보고. 시안엔 "발행본 아님" 명시.
