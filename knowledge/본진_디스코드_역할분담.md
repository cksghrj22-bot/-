# 🎛️ 본진(디스코드) ↔ 코드방 역할 분담 (2026-07-24 이찬호 지시)

> 지시: "일 밀리는 거 없이 자질구레한 것들을 디스코드(본진) 시켜라. 세 방인데도 밀린다."
> 원칙: **코드방은 창작·분석·정본관리에 집중, 본진은 코드방이 못 하거나 밀리는 실행을 전담.**
> ⚠️ 코드방 세션엔 디스코드 커넥터 없음 → 코드방이 직접 디스코드 못 보냄. **분류 + 복붙 지시문**을 만들어 이찬호가 붙여넣음.

## 🟥 본진(디스코드로 지시) 전담 — 코드방이 못 하거나 규약상 본진

| 일 | 왜 본진 |
|---|---|
| **유튜브 영상 다운로드** | 코드방 = org 프록시가 googlevideo 403 차단(2026-07-24 확정). 본진 = 프록시 밖이라 됨 |
| **발행 (유튜브·인스타·스레드)** | 규약: 발행=본진. 예약 스케줄러도 본진 |
| **아침 브리핑 / 예약 루틴·트리거** | 규약: 맥스튜디오 유일. 코드방 생성 금지 |
| **메타 토큰 발급(개발자포털)** | 데스크톱이 쉬움 + 기기신뢰. 발급 후 secrets는 본진 상시보관 |
| **대용량 영상 렌더/업로드** | 영상 원본이 본진에 있으면 본진 ffmpeg가 빠름(코드방은 드라이브 브리지 경유) |
| **큰 배치 다운로드(채널 전체 롱폼)** | 네트워크·용량 — 본진 |

## 🟩 코드방(여기) 전담 — 창작·분석·정본

| 일 | 왜 코드방 |
|---|---|
| 교육 시스템·도식·덱·상황판·보고서 | 문서 생성이 코드방 강점 |
| 파일 해독·분석 (키노트·pptx·채점표) | 바이너리 파싱·추출 |
| 드라이브 읽기·검색·카탈로그 | MCP로 목록조회·다운로드 됨 |
| 코드 커밋·정본 박제·프롬프트 관리 | 저장소가 코드방 소유 |
| 유튜브 **메타데이터**(제목·자막·썸네일) | android 우회로 조회는 됨 |

## 🌉 브리지 = 구글드라이브 「앳나운_영상」
- 코드방→본진: 코드방이 드라이브 업로드 → 본진이 읽음
- 본진→코드방: 본진이 받아 드라이브 업로드 → 코드방이 curl로 끌어 씀
- 실측: 포토싱크→「코드방_B롤_인트레이」 정상 / 재활용_소스는 유튜브 다운본 착지지점
