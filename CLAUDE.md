## 이 리포지토리의 역할

**코드의 영구 보관소 + 백업 구현.** 대화방이 사라져도 코드가 남도록 모든 구현을 여기에 커밋한다.

- `pipeline/` — 지식파이프라인 (문서 수집/청크/TF-IDF 검색 + RSS 브리핑 생성기). 브리핑 생성기는 백업용 — 실서비스 브리핑은 본진이 발신한다.
- `shorts/` — 쇼츠 자동화 (자막 ASS 생성 → ffmpeg 렌더+BGM → 유튜브/인스타 업로드). 실행은 영상이 있는 Mac에서. API 키는 `secrets/`(gitignore)에만 두고 절대 커밋하지 않는다.
- 테스트: `python3 -m unittest discover -s tests` (외부 의존성·네트워크 없이 통과해야 한다)
