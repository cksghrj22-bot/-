# 앳나운 교육자료 덱 — 젠스파크 프롬프트 (DRY & UPSTYLE 스타일 정본)

> 기준: 보미 「ATNOWN DRY & UPSTYLE ACADEMY」 키노트(2026-07 업로드). 이 디자인 언어를 커트·펌 등
> 모든 교육자료에 재사용. 젠스파크/AI 슬라이드툴에 아래 프롬프트를 붙여넣어 생성 → 인물은 AI 이미지.
> 원칙: **원본보다 더 여유있게(여백↑)·가시성 우선.**

---

## 1. 디자인 시스템 (스타일 고정값 — 모든 덱 공통)

**분위기:** 에디토리얼 럭셔리 뷰티 아카데미. 잡지 화보 같은 넉넉한 여백. 차분·고급·미니멀.

**컬러**
- 배경: 따뜻한 오프화이트/크림 `#F7F5F1`
- 본문·헤드라인: 잉크 블랙 `#111111`
- 포인트(강조 단어·라인·아이콘): 골드/탠 `#A8895E`
- 캡션·서브: 웜그레이 `#9B958C`

**타이포**
- 대제목: 우아한 세리프(Didot / Cormorant / Playfair Display 계열), 아주 큼, 자간 넉넉.
  한 제목 안에서 **일부 단어만 골드**로 (예: ATNOWN=블랙 / DRY & UPSTYLE=골드 / ACADEMY=블랙)
- 라벨·캡션: 얇은 산세리프 대문자, **자간 넓게**(letter-spacing 0.2em+), 그레이
- 한글 서브카피: 얇은 고딕, 2줄

**고정 요소**
- 좌상단 **AN 모노그램 로고**(둥근 사각 테두리 안 A·N)
- 얇은 **골드 수평 라인** + 끝에 작은 원/다이아몬드 점
- 우측 세로 라벨(예: "DRY & UPSTYLE DESIGN") 세로쓰기
- 하단 **얇은 라인 아웃라인 아이콘 줄** + 라벨 (예: DRY · BRAID · UPSTYLE · DESIGN · CAMERA · MASTER)

**인물/이미지 (AI 생성)**
- 우측 대형 **AI 인물**: 흑백/저채도 그레이스케일, 우아한 헤어(측면 프로필), 화보톤, 부드러운 조명, 배경 없음(크림에 녹아듬)
- 시술 설명 슬라이드엔 해당 기술의 헤어 클로즈업(흑백)

**레이아웃 규칙(여유있게)**
- 텍스트는 좌측 정렬, 화면의 40~50%만 사용 / 나머지는 인물·여백
- 한 슬라이드 = 한 메시지. 글머리 최대 4~5개. 폰트 크게.
- 표는 얇은 라인·넉넉한 행높이. 헤더만 골드 또는 블랙.

## 2. 덱 구조 (교육자료 공통 흐름)

1. **커버** — 아카데미명 + 한글 부제 + 아이콘 줄
2. **왜 이 과정인가** (한 문장 선언 + 근거 3)
3. **커리큘럼 한눈에** — 레벨/단계 표
4. **레벨별 상세** — 과목 + 포인트 (1레벨당 1슬라이드)
5. **핵심 기술 클로즈업** — 기술 1개당 1슬라이드(흑백 이미지 + 3포인트)
6. **평가 / 테스트 기준**
7. **마스터 — 마무리 선언**

## 3. 붙여넣기용 — 커트 덱 프롬프트

```
당신은 럭셔리 뷰티 아카데미 브랜드 덱 디자이너입니다. 아래 사양으로 16:9 슬라이드 덱을 만들어 주세요.

[브랜드] ATNOWN CUT ACADEMY (앳나운 커트 아카데미)
[한글 부제] 기본기부터 살롱워크까지 — 커트 디자이너 양성 과정
[분위기] 에디토리얼 럭셔리, 넉넉한 여백, 미니멀. 잡지 화보 톤.
[컬러] 배경 크림 #F7F5F1 / 텍스트 블랙 #111 / 포인트 골드 #A8895E / 캡션 웜그레이
[폰트] 대제목=우아한 세리프(Cormorant/Playfair), 제목 속 일부 단어만 골드 / 라벨=얇은 산세리프 대문자 자간 넓게
[고정요소] 좌상단 AN 모노그램, 얇은 골드 수평라인(끝에 점), 우측 세로라벨 "CUT DESIGN", 하단 아웃라인 아이콘 줄(SECTION·ELEVATION·LAYER·GRADATION·TEXTURE·MASTER)
[인물] 우측에 AI 생성 흑백 그레이스케일 화보 인물(단정한 커트 헤어, 측면), 배경 크림에 녹아듬. 각 기술 슬라이드엔 해당 커트 흑백 클로즈업.
[여백] 원본보다 더 여유있게 — 텍스트는 화면 좌측 45%만, 한 슬라이드 한 메시지, 글머리 4개 이하, 큰 폰트, 가시성 최우선.

[슬라이드 구성]
1) 커버
2) 왜 커트 기본기인가 — "커트는 손이 아니라 눈으로 한다" + 근거 3(도구 친숙/두상 이해/반복 숙련)
3) 커리큘럼 한눈에 — Basic 1~7 표
4~10) 레벨별 상세 (1슬라이드씩):
   Basic 1 커트 기본 — 도구와 친해지기, 빗질·가위질
   Basic 2 스퀘어 원랭스 — 두상 굴곡·Natural fall·바디포지션
   Basic 3 그라데이션(스퀘어) — 엘레베이션·오버디렉션, 바디·헤드·핑거포지션
   Basic 4 그라데이션(라운드) — 두상 굴곡·라운드 섹션
   Basic 5 레이어(Natural Inversion) — 레이어 원리·오버디렉션
   Basic 6 레이어(Flat Layer) — 두상 굴곡·헤드/바디 포지션
   Basic 7 질감처리 — 질감·양감·다양한 테크닉
11) 평가 — 반복교육으로 실전 살롱워크 자유 활용까지 완성
12) 마스터 — "같은 베이스, 다른 분위기"
```

## 4. 붙여넣기용 — 펌 덱 프롬프트

```
당신은 럭셔리 뷰티 아카데미 브랜드 덱 디자이너입니다. 아래 사양으로 16:9 슬라이드 덱을 만들어 주세요.

[브랜드] ATNOWN PERM ACADEMY (앳나운 펌 아카데미)
[한글 부제] 모질을 읽고 결과를 예측하는 — 펌·와인딩 디자이너 양성 과정
[분위기·컬러·폰트·고정요소] 커트 덱과 동일 시스템. 우측 세로라벨 "PERM & WINDING". 하단 아이콘(THEORY·WINDING·COLD PERM·HEAT PERM·DESIGN·MASTER)
[인물] AI 생성 흑백 화보 인물(웨이브/컬 헤어), 각 기술 슬라이드엔 해당 컬 흑백 클로즈업.
[여백] 원본보다 더 여유있게, 한 슬라이드 한 메시지, 가시성 최우선.

[슬라이드 구성]
1) 커버
2) 왜 펌인가 — "펌은 말기 전에 결과가 정해진다" + 근거 3(모질/연화/설계)
3) 커리큘럼 한눈에 — Lv1~5 표
4) Lv1 기초 — 모발학·ph·약제 / 머리 말리는 법 / 기본터치법 / 호리존탈 와인딩 / 세척법
5) Lv2 와인딩 4종 — 다데 · 다이애거널(사선) · 씽(Thin) · 열펌(C컬/S컬), 세척·중화·크리프
6) Lv3 콜드/특수펌 — 히피 트위스트 · 호일펌 · 스핀스왈로펌(핀컬) · 스트레이트+볼륨매직
7) Lv4 디자인 펌 — 단발·긴머리·숏 디자인, 시안 읽는 법 + 응용 스킬
8) Lv5 실전 설계 — 고객분석→상담→디자인분석→약제선택→와인딩선택+결과예측 (실모델 5회차)
9) 평가 — 실모델 회차 시행
10) 마스터 — "결과를 예측하는 손"
```
