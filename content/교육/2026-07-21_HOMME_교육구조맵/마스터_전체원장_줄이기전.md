## S2. 지금의 교육은, 홀로 서게 하는가?

## S15. B안 — 뱃지 승급 시스템
**의도(왜):** 뱃지가 시그니처 쇼의 입장권
- 레벨당 과목 3개 (선생님이 지정)
- 과목 통과 = 뱃지 도장 수집
- 코스별 뱃지 완성하면 응시 자격
- 시그니처 쇼 (디렉터 입회) 합격 시 → 디자이너 승급
- **코스별 필요 뱃지:** 플레이스 6 (업스타일 필수) · 한남 5 · 맨즈 4

## S16. B안 — 커리큘럼 아키텍처 '스스로 성장하는 서사'
**의도(왜):** 선생님은 모듈을 만들고, 학생은 경로를 만든다
- **공급 — 선생님이 모듈을 만든다:** 선생님 8명 × 레벨당 3과목 = 시그니처 모듈 / 각 과목의 '통과 기준'을 선생님이 설계
- **수요 — 학생이 경로를 만든다:** Lv.1~2 공통 기반(뱃지 고정) / Lv.2 이후 자기주도 뱃지 체인지 — 자신의 미감을 표현할 과목을 스스로 배치·선택 → 나만의 성장 경로

## S17. B안 — 뱃지 승급 진행도
- 입사 → Lv1·2 공통 기반 → 과목 선택 → 뱃지 수집 → (옴므 뱃지 4 / 한남 뱃지 5 / 플레이스 뱃지 6) → 시그니처 쇼 디렉터 입회 → 디자이너 승급
- 과목 통과 = 뱃지 1개 · 코스별 뱃지를 모으면 시그니처 쇼 응시 자격

## 소스 C. 펌 레벨 1~5 (펌 트랙)
- **1레벨**
  - 1. 기본 미용 이론(모발학) — 모발학 구조 · ph 기초 · 약제 장단점 구조 설명
  - 2. 머리 말리는 법
    - 남자(숏): 두피 먼저 건조 → 샴푸 전 머리모양 생각하며 전체 쉐입 잡고 말리기
    - 여자: 앞가리개 후 앞으로 쏟아트려 뒤→앞으로 말리는 느낌 (두피 먼저가 중요)
    - 블로우 C컬: 90% 건조 → 덴멘브러쉬 결 정리 → 섹션 → 롤 선택 → 롤 결 따라 바람 → 롤 굴리며 드라이
    - 아이롱 C컬: 100% 건조 → 덴멘브러쉬 → 섹션 → 아이롱 선택 → 200도, 아래로 반바퀴 → 뜸 → 어깨에 올려 식히기
    - 매직기 C컬: 100% 건조 → 덴멘브러쉬 → 섹션 → 매직기 선택 → 힘 빼고 공 굴리듯 부드럽게 프레스 → 식힘 중요
  - 3. 기본터치법 — 버진헤어 도포(모발에 바른다 생각, 흘러내리지 않게) · 체크(경계 약 도포 누락) · 다운펌(두상별 터치법) · 염색(원/투/쓰리 터치, 새치 커버 터치)
  - 4. 호리존탈 와인딩 특징 / 하는 법
  - 5. 업펌 + 펌 + 다운펌 세척하는 법
- **2레벨** — 이론: 다데 와인딩 / 다이애거널 / 씽 / 열펌 와인딩 이론 차이점 · 세척중화 / 크리프 사용 이유와 방법
  - 다데 와인딩(다단, Multi-base): 루트 볼륨·리프트 확보, 베이스 2~3개로 나눠 뿌리 각도/장력 유지
  - 다이애거널(사선): 얼굴 방향 자연스러운 방향성/볼륨, 섹션 사선 분할
  - 씽(Thin): 가늘고 촘촘한 컬, 얇은 섹션 균일 와인딩(약/열 균일)
  - 열펌 와인딩(디지털/셋팅, C/S컬): 연화(환원)→헹굼/전처리→와인딩→열처리→냉각→중화(산화)
- **레벨 3** — 히피 트위스트 / 호일펌 / 스핀스왈로펌(핀컬펌) 이론·차이점
  - 1. 히피트위스트 특징/원리/추천모질 2. 호일펌 3. 스핀스왈로펌+핀컬펌 4. 스트레이트+볼륨매직
- **4레벨** — 디자인펌 실습
  - 1. 단발 디자인 열펌 하는 법(디자인 하는 법) 2. 긴머리 디자인 열펌 3. 숏헤어 디자인 일반펌 4. 단발 디자인 일반펌 5. 중·긴 머리 디자인 일반펌
  - 시안 고른 후 그 시안 디자인 표현해낼 수 있는 '디자인 읽는 법' + 그 디자인 해내는 스킬 교육
- **5레벨** — 1. 고객분석 2. 펌에 대해 상담법 3. 디자인 분석 4. 약제 선택 5. 와인딩 선택+결과 예측 → 실제 모델로 고른 시안 작업 5회차 시행
