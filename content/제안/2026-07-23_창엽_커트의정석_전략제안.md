## 0. 한 줄 진단
지금 채널은 **원맨(차노) 마인드/철학**이 강점이자 리스크다. 「커트의 정석」은 채널을 **①기술 권위(authority) ②제2 얼굴(창엽) ③업계 유입(디자이너 지망생)** 으로 확장하는 첫 기둥이다. 잘 쓰면 **앳나운 = "배우러 가는 살롱"** 포지션을 선점한다.

## 1. 내 채널/브랜드의 부족점 (냉정하게)
1. **훅이 약하다.** 숏 오프닝이 정보부터 시작 → 첫 1.5초에 "어그로(도발) + 이득 약속"이 없다. 스크롤을 못 멈춘다.
2. **얼굴 브랜딩 회피.** 썸네일에서 얼굴을 뺐다(러닝은 맞았지만) — 교육/인물 콘텐츠는 **얼굴=신뢰=구독**이다. 인물 각인이 약하면 원맨 브랜드가 안 쌓인다.
3. **원맨 리스크.** 차노 한 명에 서사가 몰림 → 소재 고갈·번아웃·확장 한계. **창엽이라는 제2 페르소나**로 분산·확장이 필요.
4. **시리즈/퍼널 부재.** 낱개 영상만 있고 "EP.1→2→3" 빈지 구조·플레이리스트·구독 유도가 약하다. 숏→롱폼→예약 퍼널이 안 짜여 있다.
5. **전환(CTA) 약함.** "청담 앳나운" 위치·예약·채용으로 꽂는 마지막 한 방이 없다. 조회수가 매출/채용으로 안 내려온다.
6. **교육 authority 미확보.** 미용 "지식 재정의"는 있으나 **"기술을 가르치는 곳"** 이라는 권위 콘텐츠가 없다 → 동종업계·지망생 유입(가장 충성도 높은 시청자)이 빈다.

## 2. 「커트의 정석」 포지셔닝
- **한 문장:** "청담 앳나운이 커트의 기준을 공개한다." (교육 = 자신감의 표현 = 최고의 홍보)
- **타깃 3층:** ①일반 고객(이렇게 배운 곳이면 믿고 맡긴다=신뢰) ②디자이너 지망생/주니어(배우러 옴=채용 풀·유료교육 잠재) ③동종 원장(벤치마크=권위).
- **차노 vs 창엽 역할:** 차노=**철학·기준·브랜드 보이스**(왜 이렇게 자르는가), 창엽=**손·시연·디테일**(어떻게 자르는가). 둘이 **"기준을 말하는 사람 + 증명하는 손"** 으로 쌍을 이룬다.

## 3. 편집 방향 (메시지 우선 · 흐름 안 끊기)
- **줄기: 롱폼 교육 = 컬러·현장음 살림.** 첫/둘째줄기와 별개의 "교육 줄기"로. warm/clean 톤, 스튜디오 화이트 살리기.
- **⛔ 지식·대화는 절대 안 끊는다**(형 지시). 창엽이 기술 설명하거나 시즈들과 주고받는 구간은 통으로. 흐름=신뢰.
- **필러(무발화 이동·정리)만 부드럽게** 크로스페이드로 압축. 뚝뚝 하드컷으로 시각 흐름 깨지 않기.
- **화질 사수**(형: "너무 화질 안좋게 하지마") — 4K 소스 → 1080p crf 17~18 medium, 재인코딩 세대 최소.
- **훅 구조(교육 롱폼도):** 콜드오픈 3~6초에 **"가장 흔한 실수 한 장면 + 도발 자막"** → "정석은 따로 있다" → 본론. 인트로 길게 안 끌기.
- 자막: 교육이라 **핵심 용어·수치 강조 자막**(프리텐드). 창엽 발화 = 현장음 + 자막 병행.

## 4. 마케팅 방향 (훅·어그로·퍼널)
- **제목/훅 어그로화:** "커트의 정석"은 담백 → 썸네일·훅에서 **도발 한 스푼**. 예) "이거 모르고 자르면 6개월 고생합니다" / "미용실에서 이 말 나오면 도망치세요".
- **시리즈화:** 커트의 정석 **EP.1 (오늘)** 명시 → 다음 편 예고 → 플레이리스트. 빈지 + 구독 전환.
- **퍼널:** 숏(훅·30초 티저) → 롱폼(교육 authority) → 설명란/끝컷 **"청담 앳나운 · 예약/상담 링크 · 디자이너 지원"** CTA.
- **교차 활용:** 롱폼에서 강한 30~40초 구간 3개 뽑아 **숏 3편**으로 재편집(제작비 0, 노출 3배). 숏 하단 영어병기 + 해시태그 3개(§8-4).
- **창엽 개인 태그:** 창엽 이름/인스타를 자막·설명에 노출 → 제2 페르소나 팔로워 자산화.
