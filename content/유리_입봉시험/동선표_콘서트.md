# 🎤 유리 입봉 시험 — 콘서트 컨셉 무대 동선표 (D-1, 2026-07-17 시험)

> 컨셉: **콘서트.** 심사위원 = 관객. 입장할 때 콘서트 티켓처럼 **야키토리북(무드북/스타일북)** 배부.
> 장소: **앳나운 3층.** 양쪽 벽으로 의자 몰기 → 가운데 **워킹존** 확보.
> 무대 앞: 거울 2개를 **∧(산/역V)자**로 세팅 → 디자이너 퍼포먼스용.
> ⚠️ 표시는 유리·이찬호 확인 필요한 지점.

## 🎬 등장인물 & 셋업

| 구분 | 내용 |
|---|---|
| **유리** | 주인공 디자이너 (전 시퀀스 진행) |
| **모델 3명** | M1 = 스타일링 소개(웨딩/힙) · M2 = 펌+커트 실전 · M3 = 업스타일 실전 |
| **서브 2명** | **형우** = 음악 큐·소품·전반 서브 / **정민** = 로드 꽂기·중화·감기·말리기 |
| **음악** | 2트랙 — **A트랙**(시술·커트) / **B트랙**(워킹·모델 소개). 큐시트 아래 |
| **소품** | 야키토리북(관객 수만큼) · 면사포 · 구두 · **볼캡 · 나이키 슈즈**(체인지용) · 로드(열) · 중화제 |

**시작 직전: 모델 3명 전원 등장해 대기.** 관객에게 야키토리북 배부 완료 상태.

## 시트 1 — 오프닝 & 무드북 소개 (야키토리북)

| 순서 | 유리 | 서브 | 음악 |
|---|---|---|---|
| 1-1 | 관객에게 야키토리북(무드북) 티켓처럼 배부 → 컨셉 소개 | 형우: 배부 보조 | — |
| 1-2 | 멘트: "요즘 신부 촬영이 스튜디오 스냅에서 ⚠️(→무엇으로)넘어간다. 그 분위기에 맞춰 스타일을 바꿔봤다" | | B트랙 IN |
| 1-3 | **M1 등장** — 미니드레스(드레스 같지 않은 캐주얼) + 면사포 + 구두 → 워킹 | 형우: M1 큐 | B트랙 |
| 1-4 | **반전** — 즉석에서 **볼캡 + 나이키 슈즈**로 교체 → "요즘 이 페미니즘 + 힙의 조합 디자인이 좋다" → 다시 워킹 | 형우: 볼캡·슈즈 전달 | B트랙 |

**포인트:** 웨딩의 '정형' → 볼캡·나이키의 '힙'으로 뒤집는 게 시트1의 훅. 면사포는 유지한 채 신발/모자만 바꿔 대비 극대화.

## 시트 2 — 디자이너 퍼포먼스 (실전: 커트·펌·업스타일 병렬)

> 핵심은 **병렬 진행.** 한 모델이 중화/방치되는 시간에 다른 모델을 작업해 빈 시간을 없앤다.

| 순서 | 유리 | 서브 | 음악 |
|---|---|---|---|
| 2-1 | **M2**(펌 연화까지 끝난 상태) 커트 — **가볍게 · 사이드만 · 힙하게** + 그 부분만 설명 | | A트랙 IN |
| 2-2 | 동시에 **M3**(업스타일 준비 완료) 대기석 착석 상태 확인 | 정민: M3 대기 세팅 | A트랙 |
| 2-3 | **로드(열) 꽂기 시작** → M3 착석 → 스타일 시작(로드 다 말기) | **정민: 로드 꽂기 보조** | A트랙 |
| 2-4 | **M3 업스타일 완성** → 설명 → 워킹 | 형우: 워킹 큐 | B트랙(전환) |
| 2-5 | 이 무렵 **M2 중화 진행 중** → 유리가 설명하며 시간 확보(중화 방치 시간 활용) | 정민: M2 중화 관리 | A트랙 |
| 2-6 | M2 나와서 모델 스타일링까지 마무리 → **실전 완성** → 워킹 | 정민: 감기·말리기 | B트랙 |
| 2-7 | 새 모델(⚠️ M1 재등장? 확인) 한 번 더 워킹 | 형우: 워킹 큐 | B트랙 |

**음악 규칙:** 커트·시술 = **A트랙**, 워킹·소개 = **B트랙**. 모델 소개 순간 트랙 전환(형우 담당).
