## B. 밤작업 10 — 형님 톤 정답본 (그대로 렌더 가능) [정답본]

### B1. 왜 미용실 간 날만 예쁠까 (드라이 각도)
왜 미용실 간 날만 예쁠까요? / 집에 오면 그 머리가, 안 나오죠. / 비밀은 실력이 아니라, 각도예요. / 디자이너는 뿌리를 세워 말립니다. / 집에선 중력 방향 그대로 누르고요. / 머리는 자른 게 아니라, 세워진 겁니다. / 그날의 볼륨은, 손이 아니라 방향이 만든 거예요. / 그래서 배워야 할 건 제품이 아니라 / 말리는 방향, 딱 하나입니다.

### B2. 왜 숱을 쳤는데 더 무거워 보일까 (설계)
숱을 쳤는데, 왜 더 무거워 보일까요? / 숱치기를 빼기로만 생각해서예요. / 가위를 아무 데나 넣으면 / 겉은 그대로, 속만 비어요. / 그럼 머리가, 안에서 주저앉습니다. / 숱치기는 빼기가 아니라, 설계예요. / 어디를 남기고 어디를 비울지. / 그 각도가, 공간을 만듭니다. / 숱은 양이 아니라, 배치의 문제입니다.

### B3. 잘하는 것과 오래 하는 것은 다르다
잘하는 사람과, 오래 하는 사람은 다릅니다. / 잘하는 건, 한 번의 결과예요. / 오래 하는 건, 매일의 태도고요. / 반짝 잘하는 사람은 많아요. / 근데 십년, 이십년 같은 자리에서 / 지치지 않는 사람은, 드뭅니다. / 그건 재능이 아니라, 관리예요. / 그래서 저는 실력보다 먼저 / 오래 갈 수 있는지를, 봅니다.

### B4. 기록이란 — 방황을 멈추는 수단
서른일곱에, 매일 일기를 씁니다. / 멋있어 보이려는 게 아니에요. / 방황을 멈추려는 거죠. / 생각은 적지 않으면, 계속 맴돌거든요. / 근데 한 줄 적는 순간 / 그 고민은, 밖으로 나와요. / 밖에 나온 생각은, 정리가 됩니다. / 기록은 나를 자랑하는 게 아니라 / 어제의 나와, 겹치지 않는 방법입니다.

### B5. 어른들 말은 그 나이가 되어야 들린다
어른들 말은, 그때는 안 들렸어요. / 잔소리로만 들렸죠. / 근데 그 나이가 되니, 들립니다. / 말이 바뀐 게 아니라 / 내 귀가, 그제야 열린 거예요. / 그래서 지금 안 들리는 말도 / 버리면 안 됩니다. / 언젠가 내가, 그 나이가 되니까요. / 듣는 건 나이가 하고, 새기는 건 내가 합니다.

### B6. 미용인 차노에서, 사람 차노로
저는 머리를 자르는 사람입니다. / 근데 요즘, 생각을 더 많이 해요. / 책을 읽고, 일기를 쓰고, 달립니다. / 머리 얘기만 하던 채널에 / 왜 이런 걸 올리냐고요? / 미용은 제 일이지, 저의 전부가 아니거든요. / 좋은 디자이너이기 전에 / 저는, 생각하는 사람이고 싶어요. / 미용인 차노에서, 사람 차노로.

### B7. 실력은 손이 아니라 질문에서 자란다
실력이 손에서 자란다고, 다들 생각하죠. / 아니에요. / 질문에서 자랍니다. / 같은 시술을 십년 해도 / "왜?"를 안 물으면, 십년째 초보예요. / 손은 물어본 만큼만, 정교해집니다. / 좋은 디자이너는 손이 빠른 게 아니라 / 질문이, 깊은 사람이에요.

### B8. 상담 때 이 한마디 — "제 머리에서 뭘 보셨어요?"
미용실 상담에서, 딱 한마디만 던지세요. / "제 머리에서, 뭘 보셨어요?" / 이 질문 하나로, 디자이너가 갈립니다. / 좋은 디자이너는 여기서 / 모질, 가마, 얼굴형을 읽어내요. / 아무 말 못 하면, 아직 안 본 겁니다. / 머리가 말하는 걸, 먼저 듣는 사람. / 가격표 말고, 이 대답을 보세요.

### B9. 앳나운 사전: 공간
오늘의 단어는, 공간입니다. / 숱치기는 머리를 없애는 게 아니에요. / 사이에, 공간을 만드는 거죠. / 공간이 생기면, 공기가 들어가고 / 공기가 들어가면, 머리가 움직입니다. / 움직이는 머리가, 살아 보이는 거고요. / 볼륨은 숱이 아니라 / 비운 공간에서, 나옵니다.
