# 🧩 파이프라인 효율 확장 앱 추천 (2026-07-21)

> 기준: ①내가(Claude) API로 조종 가능한가 ②이찬호의 진짜 병목을 푸는가. 목적=자동화·글로벌·"찍기만".

## 🎁 이미 연결됨 (새로 안 사도 됨)
| 도구 | 지금 가능 |
|---|---|
| Adobe Firefly/Photoshop (MCP 연결) | 이미지 생성·누끼(배경제거)·보정·벡터화 → 카드뉴스·썸네일 |
| Canva (MCP 연결) | 템플릿 디자인 생성·내보내기 |
| ElevenLabs ✅ | 음성·다국어 더빙 |

## 🔌 붙이면 효율 큰 것 (내가 API로 조종)
| 단계 | 앱 | 병목 해소 | 임팩트 |
|---|---|---|---|
| 롱폼→쇼츠 | **Opus Clip** | 롱폼 1개→쇼츠 여러 개 자동 추출 | ⭐⭐⭐ |
| B롤 생성 | **Runway / Kling / Pika** | 못 찍는 추상·이동·마인드편 AI 생성 | ⭐⭐⭐ |
| 글로벌 | **ElevenLabs Dubbing** | 한→영/일 자동 더빙 | ⭐⭐⭐ |
| 오디오 정리 | **Auphonic** | 안경·폰 날것 음성 잡음제거·레벨 | ⭐⭐ |
| 썸네일/글자이미지 | **Ideogram / Flux** | 글자 박힌 이미지 잘 뽑음 | ⭐⭐ |
| 옛영상 화질 | **Topaz Video AI** | 리메이크 업스케일 | ⭐ |

## 👤 형님 직접(GUI, 내 직접조종 X)
| 앱 | 용도 |
|---|---|
| Descript | 롱폼 talking-head 글 편집하듯 |
| Submagic / Captions | 쇼츠 바이럴 자막 |
| CapCut | 손 다듬기 |

## 🎯 우선순위
1. **Opus Clip**(롱폼 시작 시 최고 레버리지)
2. **AI 영상생성(Runway/Kling)**(B롤 구멍)
3. **ElevenLabs Dubbing**(글로벌)
4. 이미지·디자인 = 이미 있는 Adobe/Canva로 내가 처리

## 원칙
- 앱을 많이 붙이는 게 목적 아님 — **병목 하나에 앱 하나.** 지금 병목=①롱폼 리퍼포징 ②B롤 부족 ③글로벌.
- 붙일 땐 하나씩, 효율 검증 후 다음.
