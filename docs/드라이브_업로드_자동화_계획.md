# 📤 폰 → Drive 업로드 자동화 계획 (무조건 실행) — 2026-07-21

> 목표: 찍은 영상이 **자동으로** Drive 「쇼츠_배경클립_넣는곳」에 올라가고 → A방이 B롤로 씀. 이찬호는 **찍기만.**
> 구글포토는 제외(자동싱크 없음·API막힘). 폰 카메라롤 → Drive 직행이 정답.

## 방법 비교 (아이폰)
| 방법 | 자동성 | 난이도 | 비용 | 추천 |
|---|---|---|---|---|
| **PhotoSync 앱** | ⭐ 백그라운드 자동전송 | 쉬움(5분 설정) | ₩약 6천 1회 | ✅ **1순위(무조건 됨)** |
| iOS 단축어(나이트) | 밤마다 자동 | 중(UI 헤맴 有) | 무료 | 2순위(공짜) |
| Mac + Drive 데스크톱 | 폴더 자동싱크 | Mac 필요 | 무료 | 본진 있을 때 |

## ✅ 1순위 — PhotoSync (이걸로 확정 권장)
> 아이폰 사진·영상을 **백그라운드에서 지정 클라우드 폴더로 자동 전송**하는 전용 앱. 단축어처럼 안 헤맴.
1. App Store에서 **「PhotoSync」** 설치
2. 앱 열기 → **설정(⚙️) → Configure/Add Account → Google Drive** → 로그인
3. **전송 대상 폴더 = 「쇼츠_배경클립_넣는곳」** 지정
4. **Autotransfer(자동전송) 켜기** → "새 항목만" + **백그라운드 전송 ON**
5. 필터: **동영상만** (원하면 사진 제외)
→ 끝. 이제 찍으면 알아서 그 폴더로 올라감. 신경 0.

## 2순위 — iOS 단축어 (무료, 내가 foolproof로 안내)
- 단축어 앱 → 자동화 → 시간대(예 23:00, 즉시실행 ON) → 작업: **"사진 찾기"**(동영상·오늘) → **"각 항목 반복"** → **"파일 저장"**(Google Drive/「쇼츠_배경클립_넣는곳」·저장위치 묻기 OFF)
- ⚠️ "사진 찾기"가 검색에 안 뜨면: 검색창에 **`Find Photos`**(영문) 또는 **`사진`** 만 치고 목록에서 고르기. 그래도 없으면 PhotoSync로.

## 파이프라인 측 (내가 이미 해둠)
- A방이 「쇼츠_배경클립_넣는곳」을 **매시 스캔 → B롤 카탈로그 등록 → 대본에 자동 배정** (tests/test_intake_scan.py 존재).
- 즉 **폴더에 들어오기만 하면** 나머지 자동.

## 무조건 되게 — 오늘 실행 순서
1. **PhotoSync 설치** (5분이면 세팅 끝, 제일 확실)
2. 나랑 화면 보면서 대상 폴더 「쇼츠_배경클립_넣는곳」 지정 → Autotransfer ON
3. 테스트: 영상 하나 찍고 몇 분 뒤 Drive 폴더에 들어왔나 확인
4. 되면 끝 — 이후 찍기만. 안 되면 2순위(단축어)로.

## 성공 기준
- 영상 찍음 → (아무것도 안 함) → 그 영상이 Drive 「쇼츠_배경클립_넣는곳」에 나타남 → 다음 A방 스캔 때 B롤 카탈로그 등록.
