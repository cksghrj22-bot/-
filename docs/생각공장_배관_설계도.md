## 🔧 배관 = 생각 → 다매체 자동발신 → 피드백 (스파인)

```
[형님 생각]  구술·글·영상
    ↓  ① 입력: 볼트 저장 + 음성→텍스트
[차노보이스 정제]                     내 언어로, 톤규칙 지켜
    ↓  ② 정제
[다형태 변환]  쇼츠대본 / 블로그 / 스레드 / 인스타 / 카드뉴스 / 다국어더빙 / 포토인서트영상
    ↓  ③ 변환
[매체 역제안]  "이 생각은 쇼츠+스레드가 맞다"   ← 클로드 디렉터 판단
    ↓  ④ 라우팅
[자동 발행]   유튜브 / 스레드 / 인스타 / 블로그
    ↓  ⑤ 발신
[조회수 참조 → 피드백]  잘 된 포맷·주제를 다음 제안에 반영
    ↓  ⑥ 루프백
[형님에게 피드백 + 다음 제안]
```

## 🚧 유일한 외부 블로커 = Meta 토큰 하나

- 나머지 전 구간은 배관이 있거나 내가 코드로 깐다.
- **스레드·인스타 자동발행만** Meta 토큰에 막혀 있다. 이건 형님이 컴퓨터에서 1회 발급(값 붙여넣기)해야 뚫린다.
- 그 전까지 **유튜브 축으로는 생각공장 완전자동이 오늘도 가능**하다.

## 🪣 양동이 vs 배관 구분 (원칙)
- **양동이(하지 말 것)**: 매번 손으로 파일 옮기기·1회성 처리·"이번만" 승인.
- **배관(할 것)**: 한 번 깔면 루프가 알아서 도는 코드·자동단계.
- 판단 기준: "이거 다음에도 또 손이 가나?" → 그러면 배관으로 바꾼다.
