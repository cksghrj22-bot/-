# 🏛️ AT NOWN 콘텐츠 파이프라인 — 전체 총정리 (2026-07-20)

> 지금까지 쌓아온 프로필·시스템의 단일 요약. 새 사람에게 넘기거나 새 세션이 이어받는 기준.

## 1. 정체성·미션
- **이찬호(차노쌤)** — 청담 AT NOWN 3매장(본점·플레이스·메종한남) 리더 + 크리에이터.
- **미션: "한 문장이 한 권의 책"** — 연결(한마디×자산×이질개념) → 변환(다중 포맷) → 발신(국내→글로벌).
- 브랜드 정신: **Art · Now · Own.**

## 2. 최상위 원칙
- **연동이 콘텐츠보다 먼저.** 목적(마음의 발신)이 먼저, 도구·효과는 수단. 본질이 전부(기법이 본질과 충돌하면 기법을 버린다).

## 3. 차노 보이스 (콘텐츠의 심장)
- 사고 5패턴: **재정의**(통념 뒤집기) · 작은 차이의 확대 · 판정형 질문 마무리 · 업의 철학화 · 이질 결합.
- 문체: 존댓말 단문, 훅=공감질문/단정반전, 실행 가능한 한 줄, 브랜드 선언으로 닫기. 과장·전문용어 자랑·영업멘트 금지.
- **절대 규칙: 창작 비유 금지 — 그가 실제 쓴 문장·재정의만 인용·확장. 볼트 먼저 검색해 그의 단어로.**

## 6. 제작 규격
- **쇼츠**: 1080×1920, 영상 4:5 확대크롭+상하 검정밴드, **흑백(bw)**, 제목 노랑(상단), 자막 흰글씨+검은박스(대사 최대 2줄·세그≤26자), 차노 보이스클론 TTS(일레븐랩스, 안정성0.42·스타일0.15·1.05배속·문맥 스티칭), BGM CreatorOS 피아노, 아웃트로 중앙 박스, 폰트 KyoboHandwriting2019.
- **카드뉴스「Chano's Formula」**: 실제 헤어 사진 풀블리드+다크 오버레이 / 굵은 흰 제목 / 항목 👉 값 리스트 / ✓ 쉬운정보 / 마지막장 차노 한마디. (폰트 NanumSquareRound, 이모지 폰트는 스택 맨 뒤.)
