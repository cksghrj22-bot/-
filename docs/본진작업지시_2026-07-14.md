## 1. 유튜브 업로드 OAuth (2분)

기존 유튜브 업로드 자동화에 쓰는 자격증명을 찾아 리포지토리 클론에 저장:

```bash
mkdir -p secrets
cat > secrets/youtube.json <<'EOF'
{"client_id": "<기존 client_id>", "client_secret": "<기존 secret>", "refresh_token": "<기존 refresh_token>"}
EOF
```

브라우저 업로드 방식만 있고 OAuth가 없다면: console.cloud.google.com에서 데스크톱 앱
OAuth 클라이언트 생성 → oauthplayground에서 youtube.upload 스코프로 refresh_token 발급
(브라우저 자동화로 직접 수행 가능 — 로그인 세션 이미 있음).

## 3. 메타(인스타) 장기 토큰 (15분, 브라우저 자동화로 수행)

로그인된 브라우저로 developers.facebook.com에서:
1. 앱 만들기 (유형: 비즈니스, 이름: 앳나운파이프라인)
2. Instagram Graph API 제품 추가
3. 그래프 API 탐색기 → 권한 instagram_basic, instagram_content_publish, pages_show_list → 토큰 생성
4. 액세스 토큰 디버거 → 장기 토큰으로 교환
5. me/accounts → 페이지ID → ?fields=instagram_business_account → ig_user_id 확보
6. 저장: `secrets/instagram.json` = {"access_token": "...", "ig_user_id": "..."}
7. (선택) 같은 앱에 Threads API 추가 → threads 토큰도 발급

⚠️ 발급 화면에서 막히면 스크린샷과 함께 중단 지점을 상태판에 기록 (억지로 진행 금지).
