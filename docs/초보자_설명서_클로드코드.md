# 🧒 완전 초보 설명서 — Claude Code로 영상 자동화 시작하기 (2026-07-21)

> "컴퓨터 있어야 돼?" "본진 있어야 돼?" "처음 하는 사람한테 어떻게 알려줘?"에 대한 답.
> 초등학생한테 설명하듯 쉽게. 이미 해본 사람(이찬호)이 새 사람에게 통째로 건네는 킷.

---

## 1. 제일 중요한 답 — 컴퓨터·본진 필요 없어요

**❌ 맥스튜디오(본진) 없어도 됩니다. 컴퓨터 없어도 됩니다.**

- Claude Code는 **인터넷(클라우드)** 에서 돌아가요. 내 컴퓨터가 일하는 게 아니라, **인터넷 속 컴퓨터**가 대신 일해요.
- **증거: 이찬호도 지금 폰으로 쓰고 있어요.** 영상 만들기(렌더)도 그 인터넷 컴퓨터 안에서 다 됩니다.
- 맥스튜디오(본진)는 이찬호가 "24시간 꺼지지 않고 자동으로 돌리려고" 따로 둔 것뿐. **처음 하는 사람은 필요 없어요.**

### 초등학생 버전 🧒
> Claude Code는 **"인터넷 속에 사는 만능 조수"** 예요.
> 나는 컴퓨터를 안 사도 돼요. 그냥 폰으로 조수한테 **"이런 영상 만들어줘"** 라고 말하면,
> 조수가 인터넷 안에서 뚝딱 만들어서 갖다줘요. 내가 하는 건 **말하기 + 찍기** 뿐이에요.

---

## 2. 처음 하는 사람한테 필요한 것 (딱 이것만)
| 필요한 것 | 설명 | 필수? |
|---|---|---|
| Claude 계정 | claude.ai 가입 | ✅ 필수 |
| 인터넷·폰(또는 아무 기기) | 조수한테 말 걸 창구 | ✅ 필수 |
| GitHub 계정 | 만든 코드를 저장하는 "사물함" | ✅ 필수(무료) |
| API 키 몇 개 | 목소리(TTS)·유튜브·드라이브 열쇠 | 영상까지 갈 때 |
| 맥/컴퓨터 | 24시간 무인 가동용 | ❌ 나중에·선택 |

---

## 3. 여기까지 오는 길 (초보 → 자동화, 순서대로)
1. **claude.ai/code** 접속 → Claude 로그인
2. **GitHub 연결 → 저장소(리포) 하나 만들기** (코드가 사는 집)
3. 조수한테 말하기: **"미용(내 분야) 쇼츠를 자동으로 만들고 싶어. 대본 쓰고 자막 얹어서 영상 만들어줘."**
4. 조수가 필요한 코드·폴더를 만들어줌 → 중간에 **"목소리 열쇠(API 키) 필요해요"** 하면 그때 발급해서 주기
5. **대본 쓰기 → 영상 렌더 → 결과 확인** (다 인터넷 안에서)
6. 익숙해지면 **"매시간 알아서 대본 쌓아줘"** 같은 **자동 반복(루프)** 걸기
7. B롤(배경 영상)은 **찍어서 드라이브 폴더에 올리면** 조수가 알아서 씀

> 이찬호는 여기에 더해 상황판·스레드 예약·블로그·카드뉴스까지 붙였지만, **새 사람은 3~5번만 해도 "영상 자동 생성"** 궤도에 올라요.

---

## 4. 새 사람이 붙여넣는 시작 프롬프트 (복붙용)
```
너는 내 콘텐츠 자동화 조수야. 나는 코딩 초보고, 컴퓨터 없이 폰/웹으로 쓸 거야.
[내 정보] 이름/분야/채널: (___)
[내가 원하는 것] 내 분야의 짧은 영상(쇼츠)을 반자동으로 만들고 싶어. 대본→자막→영상까지.

지켜줘:
1. 나는 초보다. 전문용어 쓰지 말고, 한 번에 한 단계씩, 내가 뭘 누르면 되는지만 알려줘.
2. 처음에 GitHub 저장소 만드는 것부터 같이 해줘. 왜 필요한지 한 줄로만 설명하고 진행해.
3. 영상 만들려면 필요한 열쇠(API 키)가 있으면, 그때그때 "이거 발급해서 나한테 줘"라고 딱 짚어줘. 미리 다 몰아서 시키지 마.
4. 내 말투·생각을 배워서 내 언어로 대본을 써. 남을 깎거나 나를 띄우는 문장은 금지.
5. 만든 건 다 저장소에 커밋해서, 대화가 사라져도 남게 해.
6. 발행(유튜브·인스타 올리기)은 내가 "올려"라고 할 때만.

시작: 먼저 나한테 물어봐 — (1) 분야가 뭐야 (2) 어떤 영상부터 만들고 싶어 (3) 지금 컴퓨터 있어 없어(없어도 됨). 답 들으면 1단계(저장소 만들기)부터 같이 하자.
```

---

## 5. 한 장 요약 (이것만 기억)
- **컴퓨터·본진 필요 없다.** 인터넷 속 조수가 대신 일한다. 폰으로 시작 가능.
- 필요한 건 **Claude 계정 + GitHub + (영상 갈 땐) 열쇠 몇 개.**
- **나는 말하고·찍기만.** 나머지(대본·자막·영상·저장)는 조수가.
- 시작 프롬프트 하나 붙여넣으면 조수가 1단계부터 손잡고 데려간다.
