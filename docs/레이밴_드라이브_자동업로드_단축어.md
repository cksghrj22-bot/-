# 📲 레이밴 → 드라이브 문방구 자동업로드 (아이폰 단축어 레시피)

> 레이밴 메타2로 찍은 영상 → 폰 사진앨범 → **구글드라이브 「앳나운_영상/문방구」 자동 업로드**.
> 올라가면 A방이 매시 스캔 → 카탈로그 → B롤 자동 배정. 이찬호는 찍기만 하면 됨.

## 사전 준비 (한 번만)
1. **파일 앱에 구글드라이브 연결**: 아이폰 「파일」 앱 → 오른쪽 위 ··· → 「편집」 → **Google Drive 켜기**.
   (구글드라이브 앱이 설치돼 있어야 함.)
2. 레이밴 영상이 **폰 사진앨범(Photos)에 들어오게** 해둔다 (Meta AI 앱에서 가져오기/자동가져오기).

## 레시피 A — 밤마다 자동 (제일 간단·추천)
「단축어」 앱 → **자동화(Automation)** 탭 → + → **개인 자동화** → **시간대(Time of Day)**
- 시각: **매일 밤 23:00** (원하면 오전에도 하나 더)
- 「즉시 실행(Run Immediately)」 **켜기** (확인 안 뜨게)

동작(Actions) 순서:
1. **사진 찾기(Find Photos)**
   - 필터: **미디어 유형 = 동영상**
   - 필터 추가: **생성일(Date Taken) = 오늘(is today)**
   - (레이밴 전용 앨범 있으면: 앨범 = 그 앨범 추가하면 더 정확)
2. **각 항목 반복(Repeat with Each)**
   3. **파일 저장(Save File)**
      - 서비스: **Google Drive**
      - 대상 폴더: **앳나운_영상 / 문방구** 로 지정
      - **저장 위치 묻기(Ask Where to Save) = 끄기** (자동 저장)

→ 밤마다 오늘 찍은 레이밴 영상이 문방구로 쏙 올라감.

## 레시피 B — Meta 앱 닫으면 자동 (실시간에 가까움)
자동화 → 개인 자동화 → **앱(App)** → **Meta AI(또는 Meta View) 닫힘(Is Closed)**
- 동작은 레시피 A의 1~3과 동일 (오늘 동영상 찾아 문방구 저장)
- 「즉시 실행」 켜기
→ 촬영 후 Meta 앱에서 확인하고 닫으면 바로 업로드.

## 중복 방지 팁
- 레시피 A는 하루 1회면 중복 거의 없음. 여러 번 돌리면 같은 영상이 "파일(1)"로 또 올라갈 수 있어 → **하루 1~2회로 제한** 권장.
- 파일명이 겹쳐도 A방 카탈로그는 파일명 기준이라 큰 문제 없음(중복분은 스킵/정리).

## 확인
업로드 후 드라이브 「앳나운_영상/문방구」에 영상이 보이면 성공. 다음 A방 매시 스캔 때 B롤 카탈로그에 자동 등록됨.
