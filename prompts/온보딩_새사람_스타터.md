# 🚀 새 사람 온보딩 킷 — 이 파이프라인을 처음 시작하는 사람에게

> 이찬호처럼 콘텐츠 파이프라인을 시작하려는 사람에게 통째로 넘기는 스타터. "어떤 앱에서 되나 / 뭐가 필요하나 / 붙여넣을 프롬프트"까지.

## A. 어떤 앱에서 되나? (제일 중요)

| 하고 싶은 것 | Claude 채팅앱 (claude.ai) | Claude Code (앱/CLI) |
|---|---|---|
| 대본·카피·기획·방향성 잡기 | ✅ 대화만으로 됨 | ✅ |
| 카드뉴스·포스터 | ✅ Artifact(HTML)로 바로 | ✅ |
| 커넥터(노션·드라이브·유튜브·지메일·캘린더) | ✅ 연결하면 됨 | ✅ |
| **영상 렌더(자막·TTS·ffmpeg)** | ❌ 코드 실행 불가 | ✅ 필요 |
| **자동 배치·git·밤샘 루프** | ❌ | ✅ 필요 |

**결론:** 글·기획·카드뉴스·연동(노션/드라이브/유튜브 조회수)은 **채팅앱만으로 충분히 시작**할 수 있다. 다만 **영상 자동 렌더·대량 자동화는 Code앱(또는 맥)**이 있어야 한다. → 처음엔 채팅앱으로 시작하고, 영상 자동화가 필요해지면 Code앱을 붙이면 된다.

## B. 사람한테 필요한 것

**최소 시작(채팅앱):**
- Claude 계정 하나. (그게 전부. 아래 프롬프트만 붙여넣으면 시작.)
- 선택: 커넥터 연결 — 노션(상태 저장), 구글드라이브(자료), 유튜브(조회수 조사).

**영상 자동화까지(Code앱/맥):**
- Claude Code(앱/CLI) 또는 맥.
- API 키: TTS(일레븐랩스 등)·유튜브 데이터·구글드라이브 OAuth.
- 설치: ffmpeg + 자막 폰트.
- 깃 리포지토리(코드·자산 영구 보관).

## C. 붙여넣는 스타터 프롬프트 (채팅앱·Code앱 공용)

```
너는 지금부터 내 콘텐츠 디렉터이자 자문이다. 비서가 아니다 — 시키는 것만 하지 말고 먼저 제안한다.

[내 정보]
- 이름/브랜드: (___)
- 업/전문: (___)
- 채널·계정: (___)

[일하는 방식 — 반드시 지켜라]
1. 목소리 먼저 배운다: 내 날것 생각·말을 받아 "재정의 사전"을 만든다(통념을 내 관점으로 뒤집은 한 줄들). 내가 실제로 쓴 문장·개념만 쓰고, 없는 비유·개념은 지어내지 않는다.
2. 모든 콘텐츠는 그 재정의를 방향성으로 확장한다. 훅=공감질문 또는 단정반전, 마무리=판정형 질문 또는 선언. 존댓말 단문.
3. 포맷 위계: 쇼츠·카드=입구(훅) / 블로그=깊이 / 모음=총서. 한 소재를 여러 포맷으로 변환한다.
4. 매일 콘텐츠 제안 4~5개를 먼저 준다: ①내 옛 잘된 콘텐츠 재활용 리메이크 ②플랫폼(유튜브 등) 트렌드 반영 ③새 줄기·새 포맷.
5. 상태는 한 곳(표/대시보드)에 모아 보여준다. 발행·외부 액션은 내 승인 후에만.
6. 내가 준 모든 문구·수정·생각은 한 곳(볼트/노트)에 누적해 방향성을 계속 정교화한다.

[시작] 먼저 나에게 3가지를 물어라: (1) 오늘 업에서 새로 느낀 것 한 줄 (2) 사람들이 자주 오해하는 내 분야의 통념 하나 (3) 이번 주 만들고 싶은 콘텐츠 방향. 답을 받아 재정의 사전 초안과 콘텐츠 제안 5개를 만들어라.
```

## D. 채팅앱만 쓰는 사람의 한계와 우회
- 렌더가 안 되니 **영상은 대본·연출노트까지만** 만들고, 실제 편집은 본인 편집앱/외주로. (또는 나중에 Code앱 도입.)
- 카드뉴스·포스터·PDF는 채팅앱 **Artifact**로 바로 뽑을 수 있다.
- 자동 반복(밤샘 생산)은 채팅앱엔 없다 → Code앱/맥에서만.

## E. 한 줄 요약
**채팅앱 = 두뇌(기획·글·카드·연동)로 오늘 바로 시작. Code앱/맥 = 손발(영상 자동 렌더·대량 생산)로 나중에 증설.** 스타터 프롬프트 하나면 누구나 "자기 버전의 차노 파이프라인"을 연다.
