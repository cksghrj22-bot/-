# 02. 쇼츠 대량생산 — 아이디어 → 촬영 가능한 대본

> 00 마스터 + 01 통과작을 입력으로 실행. 하루 10편 기본, 요청 시 100편까지 동일 규칙.

## 대본 형식 (shorts 파이프라인 .txt — 그대로 렌더러에 들어간다)

```
# 제목: (궁금하게 만드는 문장형, 25자 이내)
# 설명: (핵심 요약 + 해시태그 3~5개)
# 태그: 쉼표,구분,5개
00:00-00:02.5 훅 (공감 질문 또는 단정적 반전)
00:02.5-00:07 통념 제시 ("보통 ~라고 생각하죠")
00:07-00:17 재정의 + 증거 (보유 영상으로 보여줄 수 있는 것만)
00:17-00:26 확장 (철학 한 층 위로)
00:26-00:32 판정형 마무리 (질문 또는 실행 한 줄)
```

## 편당 규칙

- 총 28~35초. 대사 7~8줄. 한 편에 생각 하나만.
- 훅에 반드시 시청자의 언어(고객 고민)를 쓴다. 업계 용어로 훅을 열지 않는다.
- 각 대본 아래에 제작 메모를 별도 파일(제작시트)로: 원천 영상 파일명, 재컷 구간 제안, 확장 포맷.
- 시리즈 배분: 고객고민 3 : 철학 3 : 사전/교육 2 : 판매입구 2 (10편 기준 기본 비율).

## 출력·저장

- `content/shorts/YYYY-MM-DD/NN_제목.txt` 로 저장 + `제작시트.md` 갱신 + 커밋.
- 100편 모드일 때: 10편 단위로 05(검수)를 통과시키고 나서 다음 10편 생성 (불량 대량복제 방지).
