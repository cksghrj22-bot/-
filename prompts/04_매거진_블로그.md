# 04. 매거진/블로그 — 원자 → 요리된 깊이

> 반응이 검증된 원자(쇼츠·카드뉴스)만 승격시킨다. 블로그는 입구가 아니라 풀코스다.

## 구조 (완성본 「숱치기는 공간」 패턴)

1. **인사 + 공감 목록**: "이런 적 없으셨나요?" 실제 고객 언어 3~4개
2. **통념의 함정**: 다들 아는 단어의 숨은 오해
3. **재정의 선언**: 인용구 블록으로 크게 (글 전체의 축)
4. **원리 풀이**: 왜 그런가 — 비유 1개 이상 (비행기, 옷장…)
5. **작은 차이 대비**: X 생각으로 하면 → 결과 / O 생각으로 하면 → 결과
6. **달라지는 것 3가지**: 번호 목록, 고객이 얻는 것
7. **실행 한 줄**: 고객이 당장 쓸 수 있는 말/행동
8. **브랜드 선언문 + 상담 연결**

## 규칙

- 1,200~2,000자. 사진 자리 표시(> 📷)를 본문 흐름에 3~4곳 지정 — 보유 사진 파일명으로.
- 쇼츠 대본을 늘려 쓰지 말 것: 쇼츠에 없던 원리 설명·사례가 절반 이상이어야 "요리된" 것이다.
- 제목은 검색어형(고객이 검색창에 칠 문장) + 호기심형 중 검색어형 우선.
- 마무리에 관련 원자 콘텐츠 링크(쇼츠/카드뉴스) 배치 — 허브 구조.
