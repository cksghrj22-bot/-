# 07. 마스터 영상편집 프롬프트 (이제까지 쓴 모든 규칙 통합 · 2026-07-23)

> 이찬호 지시: "내가 이제까지 영상수정할 때 썼던 내용 모두 프롬프트로 가져와라."
> 쇼츠·롱폼 어떤 영상을 만들든 **이 문서 하나로 규격이 강제**된다. 개정은 이찬호만.
> 근거 원본: `knowledge/제작규격_정본.md` · `prompts/06_나레이션_라임.md` · `shorts/textsafe.py` · 대화 박제.

## 0. 대원칙 (순서대로)

1. **연동이 먼저, 콘텐츠는 그다음** (이찬호 2026-07-14).
2. **발행·외부액션은 형 컨펌 후에만.** 자가루프 상한 = 드라이브 `_최신_바로보기` 업로드 + 캘린더 발행예정 기록. **유튜브 자동 업로드 절대 금지.**
3. **막 만들지 마라** — 대본/카피는 `00_차노화_마스터` + `차노_보이스` + `키워드_지도` 거친 뒤. 원본 발화가 있으면 **그 사람 말을 그대로 살린다**(지어내지 않는다).
4. **글자 절대 안 깨지게** — 렌더 전 `shorts/textsafe.py`로 글리프·폭 검증(아래 §6).

## 2. 목소리 (일레븐랩스 · 박제)

- voice_id = **차노 프로 클론**(원본 31분). speed 1.0. stability .42 / similarity .85 / style .15 / speaker_boost.
- 60초 즉석클론 폐기.

## 3. 나레이션 라임 규약 (prompts/06 — TTS 전 무조건)

- **부호 = 낭독 지시**: 마침표 `.`=내려읽고 쉼(라임 끝 줄) / 쉼표 `,`=살짝 쉼 / 부호없음=다음 줄과 붙여 흐름.
- 종결어미·동사는 **줄 안에 붙여 끝낸다**(어중간히 끊으면 클론이 헷갈림).
- 한 줄 = 한 호흡 = 자막 한 단위.

## 4. 발음 교정 (SYNTH_FIXES · shorts/tts.py 박제)

- 잔곱슬→잔꼽쓸, 곱슬→꼽쓸, C컬→씨컬, 단발은→단바른 … (실측 STT로 확인된 것만 추가).
- 새 발음사고 나면 여기 + SYNTH_FIXES에 박제.

## 5. B롤 규칙 (⭐다양성 최우선)

- **같은 장면 두 번 금지**(다른 클립·다른 구간).
- **성희 부원장 나온 클립 금지.** 이찬호 본인/무인물/모델(시술 데모)만.
- 카탈로그 = `knowledge/broll_catalog.json`. 배치 전 `python3 -m shorts.catalog_health`로 죽은 file_id 사전차단.
- B롤 다운 = `curl "https://drive.usercontent.google.com/download?id=<fid>&export=download&confirm=t"`.
- 인트레이(코드방_B롤_인트레이) 신규 = 프레임 떠서 카테고리 분류 후 등록. 개인 일상도 형 확인되면 B롤(workout·daily).

## 6. 글자 깨짐 방지 (shorts/textsafe.py — 2026-07-23 실사고)

- **원인**: 손글씨 폰트가 '공백'을 □ 박스로 렌더(cmap엔 있다고 나오나 실제 깨짐).
- **강제**: 렌더 전 `textsafe.pick_font(text, [손글씨, 나눔])`으로 **실제 렌더 기반**(`broken_chars`) 검증 → 깨지는 폰트 자동 회피. `fit()`으로 폭 초과 자동 줄바꿈·자동 축소. **화면 밖으로 안 잘리고 두부 안 남긴다.**

## 7. 롱폼 규칙 (박제)

- **원본(DJI·인스타360 등)이 메인 화면.** 인터뷰/발화는 나레이션·자막.
- **하드컷 최소 → 크로스페이드**(xfade+acrossfade). 무음 타이틀카드로 말 끊지 않기(상단 밴드 오버레이).
- BGM 언더(5~14%) + 현장음 유지. 6분 억지로 안 채움 — **내용이 끝나면 끝.**
- **원본 발화가 있으면 그 말을 다 담는다**(텍스트 지어 얹지 않음). 러닝처럼 야외 360캠은 **바람제거 후 STT**(shorts/stt.py, highpass+afftdn) → 형 실제 말 자막.

## 8. 러닝=인생 롱폼 (2026-07-23 목적 · 이 건 전용)

- 소스: 인스타360 러닝 POV(한강 천변) + 형이 뛰며 360캠에 말한 것.
- **텍스트/나레이션 얹지 말 것.** 자막 = **형이 실제로 한 말**(STT 전사, 바람제거 필수).
- **형이 말한 것 전부 담기**(빠짐 금지). 사이 죽은 구간(무발화 러닝)만 컷 or B롤로. 6분 억지 안 채움.
- 제목 「러닝 = 인생」. 톤 warm_film(러닝은 컬러 생동감).
