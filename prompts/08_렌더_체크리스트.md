# 08. 렌더 전 필수 체크리스트 — 하라 / 하지 마라 (2026-07-15 이찬호, 무조건 통과)

> ⛔ **모든 렌더는 이 게이트를 거친다. 통과 못 하면 렌더·업로드 금지.**
> **확인은 반드시 2번** — ①프레임(화면) 눈으로 ②오디오(쉼·발음) 실측. 둘 다 OK라야 산출.
> 이찬호가 "하지 말라"고 한 건 여기 박제 = 다시는 반복 금지. 같은 키워드 실수 = 규칙 위반.

## 🚫 하지 마라 (금지 — 하나라도 있으면 폐기)

- [ ] **"저장해두세요 / 저장하세요 / 저장하고" 등 저장 CTA 절대 금지.** 마무리는 판정형 한 문장으로.
- [ ] 손글씨 딴 폰트(배민주아·감자꽃) 금지 → **교보손글씨 2019**만.
- [ ] 컬러·dim(어둡게) 금지 → **흑백, dim 없음**.
- [ ] **제목은 항상 자막보다 크게.** 길면 **2줄로(크기 유지)**, 절대 자막보다 작아지지 않는다. 화면 폭도 안 넘김.
- [ ] 배경에 **카메라 세팅·매무새 확인·쓸데없는 행동 구간** 쓰기 금지 → 멋있고 쓸만한 핵심 동작만.
- [ ] 60초 즉석 클론 금지 → **프로 클론(원본 31분)**.
- [ ] 줄마다 쉼표 남발 금지 / 마침표에서 안 쉬고 붙여읽기 금지(코드가 쉼 강제).
- [ ] 발음 오독 방치 금지(SYNTH_FIXES 확인·추가).
- [ ] **텍스트(대본) 이찬호 확인 없이 렌더 금지.**
